### 정수 입력

#### 개요
입력 받은 값을 정수으로 기록하도록 한다.

```
data = eval(input("정수를 입력하시오 : "))

data = int(input("정수를 입력하시오 : "))

data = int(input("2진수를 입력하시오 : "), 2) #2진법 숫자을 입력 받아 정수으로 data에 저장

data = int(input("8진수를 입력하시오 : "), 8) #8진법 숫자을 입력 받아 정수으로 data에 저장

data = int(input("10진수를 입력하시오 : "), 10) #10진법 숫자을 입력 받아 정수으로 data에 저장

data = int(input("16진수를 입력하시오 : "), 16) #16진법 숫자을 입력 받아 정수으로 data에 저장
```
